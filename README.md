## Prerequisites

- node 14.14.0

## Install

```sh
apt-get update -y && apt-get upgrade -y
apt-get install nodejs git
git clone https://github.com/asmin-dev/whatapp-quran-bott
cd whatapp-quran-bott
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
